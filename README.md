## APIs

Curaleaf does not publish a public developer portal, REST API, or partner API documentation. The consumer site is gated by an age-verification flow and the business operates as a state-licensed cannabis retailer rather than a developer platform. Any third-party integrations (e-commerce, dispensary POS, compliance/seed-to-sale, loyalty) are arranged through partners and vendors rather than a self-service API program.
